# Proyek-Web-Tugas-1
Pada tugas HTML pertama ini, saya hanya menggunakan perintah perintah basic yang diajarkan seperti, heading dengan kode h1, h2, dst yang akan semakin kecil ukuran headingnya bersamaan dengan besarnya angka dibelakang h. Kemudian juga ada paragraph dengan kode p, sehingga teks tidak dalam bold dan cocok untuk penjelasan rinci. Kemudian juga terdapat kode table yang disini saya gunakan untuk menyimpan data pendidikan saya yang berisi tahun, instansi, dan juga jurusannya.

Kemudian saya juga menggunakan kode list, unordered list untuk hobbies dan ordered list untuk games yang dimana saya set dimulai dengan angka 1. Kemudian saya juga membuat form section dari kode form action untuk memberikan kesan interaktif kepada pembaca halaman web saya.
